Club Golf Website

Link:

csufclubgolf.org

Project Description:

This is a website I created for the CSUF Club Golf team. The website was created using HTML, CSS, and JavaScript. It includes information about the club, upcoming events, pictures, and a contact form.

Project Features:

  • Responsive design for mobile and desktop
  • Contact form that sends an email to the club president
  • Gallery of pictures from past events
  • Information about the club and upcoming events
  • Links to social media accounts

Languages Used:

  • HTML
  • CSS
  • JavaScript

Problems Faced:

One of the main challenges I faced was teaching myself JavaScript and CSS. I had prior experience building simple websites, but it was very basic. However, with a few YouTube videos I was able to make the website look nicer. I also had to teach myself the process of deploying the website. From buying the domain to hosting the website, I had to learn how to do everything.
