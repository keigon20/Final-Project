Toy Compiler

This is a toy compiler that I created as part of my coursework. It is designed to take C++ and compile it into machine code.

The compiler is written in C++ and uses a recursive descent parser to parse the input code. It generates an abstract syntax tree (AST) and then traverses the AST to generate machine code.

Features

  • Recursive descent parser
  • Abstract syntax tree (AST) generation
  • Machine code generation
  • Support for basic C++ constructs
  • Simple error handling

Languages Used:

  • C++

Problems Faced:

One of the main challenges I faced was understanding the theory behind compilers and how to implement a recursive descent parser. We spent hours debugging edge cases. However, my team worked really well together delegating responsibilities and helping each other out. I learned a lot about how compilers work and how to implement one from scratch.
